## 📂 重要文件夹详解

### 1. **src/** - 源代码文件夹 ⭐⭐⭐ 最重要！

**这是什么？**
这是你写代码的地方，所有.java文件都放在这里。

**为什么叫src？**
src是"source"（源代码）的缩写。

**怎么用？**
- 新建Java类：右键src → New → Java Class
- 新建包：右键src → New → Package

**标准项目结构：**
```
src/
├── com/
│   └── example/
│       └── practice/           # 包名（公司域名倒写）
│           ├── Main.java        # 主类
│           ├── Student.java     # 学生类
│           └── Teacher.java     # 老师类
```

### 2. **.idea/** - IDEA配置文件夹

**这是什么？**
IntelliJ IDEA的工作配置，记录：
- 你的项目设置
- 代码风格
- 运行配置
- 等等...

**重要提示：**
- ✅ 这个文件夹是IDEA自动管理的
- ❌ 不要手动修改里面的文件
- ❌ 不要提交到Git（已经在.gitignore中）

### 3. **out/** - 编译输出文件夹

**这是什么？**
你写的.java文件会被编译成.class文件，放在这里。

**新手理解：**
- .java = 源代码（人类看得懂）
- .class = 字节码（JVM看得懂）
- out文件夹存放所有的.class文件

**重要提示：**
- ✅ IDEA自动管理
- ❌ 不要手动修改
- ❌ 不要提交到Git

### 4. **JavaPractice.iml** - 项目配置文件

**这是什么？**
IDEA的项目模块配置文件，记录：
- JDK版本
- 依赖库
- 模块信息

**新手理解：**
类似于"项目说明书"，告诉IDEA这个项目的基本信息。

**重要提示：**
- ✅ IDEA自动管理
- ✅ 可以提交到Git（其他开发者用IDEA打开时能识别）

### 5. **.gitignore** - Git忽略配置

**这是什么？**
告诉Git哪些文件不需要提交到版本控制。

**为什么需要？**
有些文件是IDEA自动生成的，每个人都不一样，不需要共享。

**常见忽略内容：**
```
# IDEA配置
.idea/
*.iml

# 编译输出
out/

# 日志文件
*.log

# 临时文件
*.tmp
*.bak
```

## 📝 包（Package）的概念

### 什么是包？

**想象一下图书馆：**
- 没有分类：所有书堆在一起，找不到想要的
- 有分类：文学类、科技类、历史类...容易找到

**Java包的作用：**
- 组织类（像图书馆分类）
- 避免命名冲突（两个人都可以叫Student类）
- 访问控制

### 包的命名规则

```
公司域名倒写.项目名.模块名

例如：
com.example.practice     # 练习项目
com.taobao.web           # 淘宝Web模块
com.baidu.search         # 百度搜索模块
```

### 在IDEA中创建包

1. **方法一**：新建类时指定包名
   - 右键src → New → Java Class
   - Name: `Student`
   - Package: `com.example.practice.entity`

2. **方法二**：先创建包，再创建类
   - 右键src → New → Package
   - Name: `com.example.practice.entity`
   - 右键新建的包 → New → Java Class
   - Name: `Student`

## 🚀 你的第一个练习

现在让我帮你创建第一周的练习文件。

### 练习1：变量练习

在src目录下创建`VariablePractice.java`：

```java
public class VariablePractice {
    public static void main(String[] args) {
        // 整数类型
        int age = 20;
        long population = 1400000000L;  // 注意L后缀

        // 小数类型
        double price = 99.99;
        float interestRate = 0.035f;    // 注意f后缀

        // 字符和字符串
        char grade = 'A';
        String name = "张三";

        // 布尔类型
        boolean isStudent = true;

        // 输出
        System.out.println("姓名：" + name);
        System.out.println("年龄：" + age);
        System.out.println("是否是学生：" + isStudent);
    }
}
```

### 练习2：条件判断练习

创建`ConditionPractice.java`：

```java
import java.util.Scanner;

public class ConditionPractice {
    public static void main(String[] args) {
        // 创建Scanner对象，用于接收用户输入
        Scanner scanner = new Scanner(System.in);

        System.out.print("请输入你的成绩：");
        int score = scanner.nextInt();

        // if-else判断
        if (score >= 90) {
            System.out.println("优秀！");
        } else if (score >= 80) {
            System.out.println("良好！");
        } else if (score >= 60) {
            System.out.println("及格！");
        } else {
            System.out.println("不及格，继续努力！");
        }

        scanner.close();
    }
}
```

### 练习3：循环练习

创建`LoopPractice.java`：

```java
public class LoopPractice {
    public static void main(String[] args) {
        // for循环：打印1到10
        System.out.println("=== for循环打印1-10 ===");
        for (int i = 1; i <= 10; i++) {
            System.out.print(i + " ");
        }
        System.out.println();  // 换行

        // while循环：计算1+2+3+...+100的和
        System.out.println("\n=== while循环求和 ===");
        int sum = 0;
        int i = 1;
        while (i <= 100) {
            sum += i;
            i++;
        }
        System.out.println("1+2+3+...+100 = " + sum);

        // 嵌套循环：打印九九乘法表
        System.out.println("\n=== 九九乘法表 ===");
        for (int j = 1; j <= 9; j++) {
            for (int k = 1; k <= j; k++) {
                System.out.print(k + "×" + j + "=" + (j * k) + "\t");
            }
            System.out.println();
        }
    }
}
```

## 💡 新手常见问题

### Q1: 为什么要用包？

**A**: 
- 组织代码，像文件柜一样分类存放
- 避免同名冲突，两个包都可以有Student类
- 访问控制，某些类只能在包内访问

### Q2: .class文件是什么？

**A**: 
- Java源代码编译后生成的字节码文件
- JVM（Java虚拟机）执行的是.class文件
- 你不需要关心它，IDEA会自动处理

### Q3: out文件夹可以删吗？

**A**: 
- 可以删，IDEA会重新生成
- 但不建议删，每次重新编译需要时间

### Q4: 如何运行程序？

**A**: 
- 方法1：点击代码行旁边的绿色三角▶️
- 方法2：右键代码编辑区 → Run '类名'
- 方法3：快捷键 Shift + F10

### Q5: 程序出错了怎么办？

**A**: 
- 看红色波浪线，IDEA会提示错误
- 把鼠标放在红线上，看错误提示
- 按Alt + Enter，IDEA会给出修复建议
